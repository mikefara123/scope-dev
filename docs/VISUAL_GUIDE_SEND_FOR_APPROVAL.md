# 📧 Visual Guide: How to Send a Budget for Approval

## ⚠️ IMPORTANT: You're Currently on the Wrong Page!

The screenshot you shared shows the **"All Budgets"** page (the list/overview page).

The **"Send for Approval"** button is NOT on this page.

---

## ✅ Here's What You Need to Do:

### **STEP 1: Click "Edit" on Any Budget Card**

On the current page you're viewing, you'll see budget cards that look like this:

```
┌─────────────────────────────────────────┐
│  Initial Budget v1.2          [⋮]       │
│  Johnson Residence                      │
│  Phase 1 - Living Areas                │
│                                         │
│  [Approved]                             │
│                                         │
│  Total Amount:        $51,628           │
│  Items:                     5           │
│                                         │
│  Updated 9 days ago                     │
│                                         │
│  [   Edit   ]  [   View   ]            │
└─────────────────────────────────────────┘
```

👉 **Click the "Edit" button** (outlined button on the left)

---

### **STEP 2: Budget Builder Opens**

After clicking "Edit", you'll be taken to the **Budget Builder** page.

This page looks like a spreadsheet/table with budget line items.

---

### **STEP 3: Look at the Top Toolbar**

At the very top of the Budget Builder page, you'll see this toolbar:

```
[← Back]  Budget v1.2 - Phase 1   [Status Badge]   [Save]  [Preview]  [📧 Send for Approval]
```

👉 **Click "Send for Approval"** (teal/secondary color button with envelope icon)

**OR**

Look at the **Bottom Bar** (below the budget grid):

```
Total Items: 5    Subtotal: $45,000    Tax: $3,200        Grand Total: $51,628    [📧 Send for Approval]
```

👉 **Click "Send for Approval"** (large button on the bottom right)

---

### **STEP 4: Configure & Send**

The "Send for Approval" modal will pop up where you can:

1. ✅ Add customer email(s)
2. ✅ Add CC recipients (optional)
3. ✅ Write a custom message
4. ✅ Configure display options
5. ✅ Click "Send for Approval" button

---
